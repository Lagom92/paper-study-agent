# 논문 학습 에이전트 with LangGraph 📚🤖



<br/>

Streamlit 실행 방법

```cmd
streamlit run app.py
```



<br/>

현재 기능

1. 논문 PDF 파일 1개를 업로드
2. GPT를 이용한 챗봇을 통해 질의응답

추후 수정사항

4. 속도가 너무 느림
